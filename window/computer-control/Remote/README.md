# 远程控制

## 实现功能
* 远程CMD
* 远程进程
* 远程窗口
* 远程桌面
* 远程通话
* 远程视频
* 远程文件
* 远程注册表

## 文件结构
* Remote 文件夹是控制端的代码，采用VS2010开发。
* ServerDll 文件夹中是被控端的代码，采用VC6.0开发。

## 关于
基于开源的远控木马gh0st，主要用于学习Windows下的应用程序开发。涉及很多Windows开发技术，例如IOCP，文件操作，
线程同步等等，是非常好的学习Windows平台下网络通信与MFC开发的项目。在开发的整个过程中更能体会到C++
对于大型程序开发的驾驭能力。
